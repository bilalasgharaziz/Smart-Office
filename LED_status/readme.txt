Getting Started
---------------
To run this code, you need to have Aurdino Ide installed on your machine. Additionally, you should install the required libraries and Boards mentioned in requirements.txt

Firebase Setup
--------------
Make sure the Firebase Realtime Database is up and running and have the necessary permissions to access Firebase.

Data Retrieving
--------------
It reads the sensor readings stored in Firebase RTDB, based on the retrieved value, occupancy status is shown on the RGB LED.

Error Handling
--------------
It includes mechanisms to handle potential errors during data transmission to Firebase, providing informative error messages.
